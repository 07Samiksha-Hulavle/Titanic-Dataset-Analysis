# Titanic-Dataset-Analysis

# Project Overview
Exploratory Data Analysis on Titanic dataset to identify key factors affecting passenger survival using Python and data visualization.

# Tools Used
- Python
- Pandas
- seaborn
- Matplotlib
- Google Colab

# Key Insights
- Females had higher survival rates than males
- 1st class passengers had better survival chances
- Class and gender strongly influenced survival
